# Aurelia UI Framework


* Running examples

  `yarn start`

* Building the plugin

  `yarn dist`

* Using plugin as local link

  `aurelia-ui-framework$> yarn link`

  `project$> yarn link aurelia-ui-framework`

  > webpack.config `resolve.symlinks = false`
